## Overview

This package is part of the Coze Studio monorepo and provides workflow functionality. It includes component, hook, adapter and more.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@coze-workflow/variable": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
