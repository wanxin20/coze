## Overview

This package is part of the Coze Studio monorepo and provides architecture functionality. It includes component, editor.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@coze-workflow/sdk": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
