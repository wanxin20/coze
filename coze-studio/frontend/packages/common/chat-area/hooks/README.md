## Overview

This package is part of the Coze Studio monorepo and provides utilities functionality. It includes hook.

## Getting Started

### Installation

Add this package to your `package.json`:

```json
{
  "dependencies": {
    "@coze-common/chat-hooks": "workspace:*"
  }
}
```

Then run:

```bash
rush update
```
